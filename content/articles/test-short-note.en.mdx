This is a deliberately short layout test fixture — not real published writing. It exists so
the Field Notes list card and detail page can be checked against a minimal body: no headings,
no lists, no code, just a few plain paragraphs.

A short note should still feel calm and readable. Spacing, type size, and the footer meta
row need to hold up when there is almost nothing else on the page. If this looks cramped or
sparse in a strange way, the article chrome is doing too much work for too little content.

Keep this file as `draft` so it never appears in public listings or the sitemap. Review it
locally via its direct URL when iterating on article typography and empty-ish states.
